7. Endpoints da API

7.1. Autenticação (/auth) [cite: 19]

Método Rota Proteção JWT Descrição
POST /auth/login Não Login de usuário. [cite: 20, 22]
POST /auth/refresh Não [cite: 23] Atualização de token. [cite: 23]

7.2. Usuários (/user)

Método Rota Proteção JWT Acesso
POST /user/register Não [cite: 27] Público [cite: 27]
POST /user/booking Sim Visitor, Organizer, Admin [cite: 29]
GET /user/ Sim Admin [cite: 31]
GET /user/:id Sim Admin [cite: 33]
PUT /user/:id Sim Todos (restrito ao próprio usuário ou Admin) [cite: 35]
DELETE /user/:id Sim [cite: 36] Admin [cite: 36]

7.3. Locais (/place) [cite: 37]

Método Rota Proteção JWT Acesso
POST /place/ Sim Organizer, Admin [cite: 40]
GET /place/ Sim [cite: 41] Todos [cite: 41]
GET /place/:id Sim [cite: 42] Todos [cite: 43]
PUT /place/:id Sim Organizer, Admin [cite: 44]
DELETE /place/:id Sim [cite: 45] Admin [cite: 45]

7.4. Eventos (/event) [cite: 46]

Método Rota Proteção JWT Acesso
POST /event/ Sim Organizer, Admin [cite: 49]
GET /event/ Sim [cite: 50] Todos [cite: 50]
GET /event/:id Sim [cite: 51] Todos [cite: 52]
PUT /event/:id Sim Organizer, Admin [cite: 53]
DELETE /event/:id Sim [cite: 54] Organizer, Admin [cite: 54]

7.5. Visitas Realizadas (/visited)

Método Rota Proteção JWT Acesso
POST /visited/ Sim [cite: 58] Todos [cite: 59]
GET /visited/ Sim [cite: 61] Todos (Admin vê tudo) [cite: 61]
GET /visited/user/:id Sim [cite: 62] Visitor/Admin/Organizer [cite: 62]
GET /visited/place/:id Sim [cite: 63] Todos [cite: 63]
PUT /visited/ Sim [cite: 65] Todos [cite: 65]
DELETE /visited/ Sim [cite: 67] Organizer, Admin [cite: 67]

8. Exemplos de Requisição

8.1. Criar Local [cite: 68]

POST /place
Content-Type: application/json

{
  "name": "Estação das Docas",
  "type": "cultural",
  "address": {
    "postalColde": "66010-020", 
    "street": "Av. Boulevard Castilhos França",
    "numberHouse": 600,
    "complement": "Armazém 2"
  }
}

8.2. Criar Evento [cite: 69]

POST /event
Content-Type: application/json

{
  "title": "Tour COP-30",
  "eventDate": "2025-07-15T14:00:00.000Z",
  "description": "Tour pelo centro histórico de Belém.",
  "place": "UUID-do-place"
}

8.3. Agendar Visita [cite: 70]

POST /user/booking
Content-Type: application/json

{
  "eventId": "UUID-event",
  "userId": "UUID-user"
}

8.4. Registrar Visita

POST /visited
Content-Type: application/json

{
  "userId": "UUID-user",
  "placeId": "UUID-place",
  "visitDate": "2025-05-24T10:30:00.000Z"
}
