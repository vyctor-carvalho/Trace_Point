1. Visão Geral do Projeto

O Trace Point - Backend é uma API RESTful robusta e escalável, projetada para gerenciar visitas e fluxo de participantes durante a COP-30 em Belém, PA. [cite: 1]

A solução permite[cite: 2]:

  • Registro de usuários com diferentes níveis de acesso (Administrador, Visitante, Organizador)[cite: 2];
  • Criação e gerenciamento de eventos e locais[cite: 2];
  • Agendamento de visitas, proporcionando organização e eficiência[cite: 3].

2. Missão do Projeto

Ser a principal plataforma de referência na gestão inteligente de visitas, eventos e pontos turísticos em grandes conferências e eventos internacionais, proporcionando controle, segurança e experiência eficiente para todos os participantes.

Ademais, desenvolver uma solução tecnológica escalável, segura e intuitiva que[cite: 4]:

  • Facilite o planejamento e acompanhamento de visitas durante a COP-30[cite: 4];
  • Promova a valorização de pontos culturais, turísticos e ambientais de Belém[cite: 5];
  • Otimize a logística de recepção de milhares de visitantes com dados em tempo real[cite: 6];
  • Apoie os organizadores e autoridades com informações estratégicas para tomada de decisão[cite: 7].

3. Tecnologias Empregadas

  • Node.js 18: Ambiente de execução assíncrono para aplicações de alta performance. [cite: 8]
  • TypeScript: Superset do JavaScript com tipagem estática. [cite: 9]
  • PostgreSQL: Banco de dados relacional open-source robusto.
  • TypeORM: ORM para integração com PostgreSQL usando TypeScript. [cite: 10]
  • Docker + Docker Compose: Conteinerização da aplicação e banco.
  • pgAdmin: Interface gráfica para gerenciar o banco PostgreSQL. [cite: 11]
