Guia de Instalação e Configuração

4. Requisitos de Ambiente

4.1. Com Docker:

  • Docker
  • Docker Compose

4.2. Execução Manual:

  • Node.js 18+
  • PostgreSQL
  • NPM

5. Instalação e Configuração

5.1. Clonagem do Repositório

git clone https://github.com/vyctor-carvalho/Trace_Point.git
cd Trace_Point

5.2. Rodando com Docker (Recomendado)

Copie o arquivo de ambiente:

cp .env.copy .env

Edite as variáveis no arquivo .env conforme necessário. Para o Docker, um exemplo de configuração seria:

DB_HOST=db
DB_PORT=5432
DB_USER=postgres
DB_PASSWORD=postgres
DB_NAME=tracepoint

Suba os containers:

docker-compose up --build

5.3. Acesso via pgAdmin (com Docker)

Acesse o pgAdmin (geralmente em http://localhost:8080 ou a porta que você configurou para o container do pgAdmin) e conecte-se ao servidor do banco de dados com as seguintes credenciais:

  • Host: db
  • Porta: 5432
  • Usuário: postgres
  • Senha: postgres

5.4. Execução Manual

Instale as dependências:

npm install

Configure seu arquivo .env manualmente com as credenciais do seu banco de dados PostgreSQL local e outras variáveis necessárias (como SYSTEM_API_PORT, JWT_SECRET, etc.).

Rode as migrações do banco de dados:

npm run migration:run

Inicie o servidor de desenvolvimento:

npm run dev
